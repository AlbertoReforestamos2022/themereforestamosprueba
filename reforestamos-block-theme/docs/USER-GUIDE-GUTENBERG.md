# Guía de Uso del Editor Gutenberg — Reforestamos México

## Introducción

El sitio de Reforestamos México utiliza el editor de bloques de WordPress (Gutenberg) para crear y editar contenido. Esta guía explica cómo usar el editor y los bloques personalizados disponibles.

---

## Acceder al Editor

1. Inicia sesión en el panel de administración: `https://reforestamos.org/wp-admin`
2. Ve a **Páginas** o **Entradas** en el menú lateral
3. Haz clic en **Añadir nueva** o edita una existente
4. El editor de bloques se abrirá automáticamente

---

## Conceptos Básicos

### ¿Qué es un Bloque?

Cada elemento de contenido es un "bloque": un párrafo, una imagen, un encabezado, un video, etc. Los bloques se apilan verticalmente para formar la página.

### Agregar un Bloque

1. Haz clic en el botón **+** (insertar bloque) en la barra superior o entre bloques
2. Busca el bloque que necesitas por nombre
3. Los bloques personalizados de Reforestamos están en la categoría **"Reforestamos"**
4. Haz clic en el bloque para insertarlo

### Mover Bloques

- Usa las flechas ↑ ↓ que aparecen al seleccionar un bloque
- O arrastra el bloque usando el ícono de 6 puntos (⠿)

### Eliminar un Bloque

1. Selecciona el bloque
2. Haz clic en los tres puntos (⋮) de la barra de herramientas del bloque
3. Selecciona **Eliminar bloque**

## Bloques Personalizados de Reforestamos

### Hero Section
Sección principal con imagen de fondo y texto destacado.
- **Título**: Texto principal grande
- **Subtítulo**: Texto secundario
- **Imagen de fondo**: Selecciona desde la biblioteca de medios
- **Botón**: Texto y URL del botón de acción
- **Opacidad del overlay**: Controla la oscuridad sobre la imagen (panel lateral)

### Carousel
Carrusel de imágenes con transición automática.
- **Imágenes**: Agrega múltiples imágenes desde la biblioteca
- **Autoplay**: Activa/desactiva la transición automática
- **Intervalo**: Tiempo entre transiciones (en milisegundos)

### Cards de Enlaces
Tarjetas con enlaces a otras secciones o páginas.
- **Tarjetas**: Agrega título, descripción, imagen y URL para cada tarjeta
- **Columnas**: Número de columnas en el grid (2, 3 o 4)

### Cards de Iniciativas
Tarjetas especiales para mostrar iniciativas de reforestación.
- **Iniciativas**: Agrega título, descripción e imagen para cada una
- **Layout**: Selecciona entre diferentes disposiciones

### FAQs (Preguntas Frecuentes)
Acordeón de preguntas y respuestas.
- **Preguntas**: Agrega pares de pregunta/respuesta
- **Abrir primera**: Muestra la primera pregunta abierta por defecto
- **Permitir múltiples**: Permite abrir varias respuestas a la vez

### Timeline (Línea de Tiempo)
Línea de tiempo vertical para mostrar hitos o eventos.
- **Eventos**: Agrega fecha, título y descripción para cada punto
- **Orientación**: Vertical u horizontal

### Documents (Documentos)
Lista de documentos descargables.
- **Documentos**: Agrega archivos PDF, DOC, etc. desde la biblioteca
- **Categoría**: Filtra por categoría de documento
- **Ordenar por**: Nombre, fecha o tipo

### Galería con Tabs
Galería de imágenes organizada por pestañas/categorías.
- **Galerías**: Crea pestañas con nombre y agrega imágenes a cada una
- **Tab por defecto**: Selecciona qué pestaña se muestra primero

### Logos de Aliados
Grid de logos de organizaciones aliadas.
- **Logos**: Agrega imágenes de logos
- **Columnas**: Número de logos por fila
- **Enlazable**: Activa para que los logos sean clicables

### Sobre Nosotros
Sección completa con información del equipo y estadísticas.
- **Título y contenido**: Texto descriptivo
- **Equipo**: Agrega miembros con foto, nombre y cargo
- **Estadísticas**: Números destacados (árboles plantados, hectáreas, etc.)

### Eventos Próximos
Lista automática de próximos eventos de reforestación.
- **Cantidad**: Número de eventos a mostrar
- **Mostrar pasados**: Incluir eventos ya realizados
- **Layout**: Lista o grid

### Contacto
Formulario de contacto integrado.
- **ID del formulario**: Identificador del formulario
- **Mostrar teléfono**: Incluir campo de teléfono
- **Mostrar dirección**: Incluir campo de dirección

### Texto e Imagen
Sección de dos columnas con texto e imagen.
- **Contenido**: Texto con formato
- **Imagen**: Selecciona desde la biblioteca
- **Posición de imagen**: Izquierda o derecha

### Lista Personalizada
Lista con iconos personalizados.
- **Items**: Agrega elementos de lista
- **Icono**: Selecciona el icono para los bullets
- **Estilo**: Selecciona el estilo visual

## Panel de Configuración (Sidebar)

Al seleccionar un bloque, el panel lateral derecho muestra opciones adicionales:

- **Configuración del bloque**: Opciones específicas del bloque seleccionado
- **Estilos**: Colores, tipografía y espaciado
- **Avanzado**: Clases CSS adicionales y ancla HTML

---

## Patrones de Bloques

Los patrones son combinaciones pre-diseñadas de bloques que puedes insertar rápidamente:

1. Haz clic en **+** para insertar
2. Ve a la pestaña **Patrones**
3. Busca en la categoría **Reforestamos**
4. Patrones disponibles:
   - **Hero Section**: Sección hero completa con imagen y botón
   - **Sección de Contacto**: Formulario con información de contacto
   - **Equipo**: Grid de miembros del equipo
   - **Testimonios**: Sección de testimonios
   - **Footer Completo**: Pie de página con columnas

---

## Guardar y Publicar

- **Guardar borrador**: Guarda sin publicar (botón superior derecho)
- **Vista previa**: Ve cómo se verá la página antes de publicar
- **Publicar**: Hace la página visible para todos los visitantes
- **Actualizar**: Guarda cambios en una página ya publicada

---

## Consejos

- Usa **Ctrl+Z** (o **Cmd+Z** en Mac) para deshacer cambios
- Usa el **Modo lista** (ícono de 3 líneas) para ver la estructura de bloques
- Arrastra imágenes directamente desde tu computadora al editor
- Usa **/** seguido del nombre del bloque para insertar rápidamente (ej: `/hero`)

---

## Soporte

Si tienes problemas con el editor, contacta al equipo de desarrollo o consulta la [documentación de WordPress sobre el editor de bloques](https://wordpress.org/documentation/article/wordpress-block-editor/).
